# High-resolution-Synthetic-Dataset-for-China-s-Power-System
The document contains 5 Excel spreadsheets.  
The 'Gen.xlsx' records diverse generators data as of the end of 2024, including coal-fired, gas, biomass, hydropower, nuclear, solar PV, wind, battery storage, and PHS.  
  The attributes of each generator are provided.  
The actual installed capacity of different generation-technologies in each province from 2011 to 2024 is also recorded.

The 'Gen_cost.xlsx' contains four sheets. Among them, 'coal_fired_cost' records the secondary cost coefficients of coal-fired generation units for different years, as well as the coal consumption for start-up and shut-down. 'Other_gencost' records the cost per kilowatt-hour for gas-fired, biomass, and nuclear generation units. 'coal_price' is the preprocessed data showing the annual average coal prices for each province from 2011 to 2024. 'coal_price_raw' records the raw coal price data.  

The 'Hydro_timeseries.xlsx' records the monthly capacity factors of the hydropower generation units from 2011 to 2024.  

The 'Lines.xlsx' records the inter-provincial ultra-high voltage transmission lines that have been put into operation up to 2024 and their attributes.  

The 'Backup_cost' records the backup cost price for each province.  

The 'Load' file includes load time series for each province from 2011-2024.  

The 'Visualization' file include the codes, maps, and data for generating figures in the paper.
